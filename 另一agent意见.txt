  🔴 严重问题 (5项 — 应优先修复)

  1. services/ai.py 缺少 import os

  # 第180行使用了 os.path.join / os.path.dirname / os.path.abspath
  # 但模块顶部没有 import os → 文档知识库注入静默失败
  影响： 所有本地知识库文档（美食攻略、娱乐指南）注入AI提示词的功能完全失效。

  2. AI截断检测逻辑错误

  # services/ai.py:875
  truncated = stop in ('max_tokens', 'length', 'stop')
  # 'stop' 是自然完成，不是截断 → 正常回复尾部错误提示「继续」

  3. _vary_reply_opener() 从未被调用

  AI回复多样性（7种开场白轮转）的函数已定义但从未调用，使得个性化回复功能形同虚设。

  4. BnB色彩架构 — 60+处hex硬编码

  static/css/style.css 中 hero/page-header/footer 等区域的渐变色大量使用硬编码 hex（如 #4a7c59、#8eb8c9）。新增BnB或调整色彩时这些值不会被CSS变量覆盖，导致UI不一致。违反CLAUDE.md中声明的"组件样式零硬编码"原则。

  5. _get_bnb_id 在8个服务文件中完全重复

  # 以下文件各有一份完全相同的实现：
  services/orders.py, menu.py, travel.py, rooms.py,
  tea.py, quick.py, notify.py, healing.py
