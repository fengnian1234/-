# 庐山牯岭镇美食全攻略（2026版）

## 一、必吃餐厅（按优先级排序）

### 1. 璐姑娘餐厅 ⭐强烈推荐
- **地址**：牯岭正街47号（清真寺斜对面，观光车直达）
- **人均**：60-90元
- **荣誉**：2023-2025连续3年"年度星级商户"，九江农家菜热门榜第3名
- **招牌菜**：庐山特产石鸡、牯岭牛骨捞粉、庐山小柴鸡、干锅鹿茸菇、泉水香味鱼片、云雾茶香骨、钵钵猪脚
- **特点**：包厢/可带宠物/宝宝椅/有节目表演，本地人也常去，非一次性买卖
- **适合**：游客尝鲜、家庭聚餐、情侣约会

### 2. 石牛酒家 ⭐本地人首选
- **地址**：牯岭镇正街（街心花园对面）
- **人均**：55-75元
- **荣誉**：庐山当地风味餐厅榜No.1
- **招牌菜**：三杯鸡、土豆烧牛肉、黄豆烧皖鱼、野芹菜、石耳烧鸡、石鸡烧石鱼
- **特点**：传统赣菜老店，现点现炒无预制菜，分量扎实，烟火气最浓
- **适合**：追求地道本土味、平价实惠、多人聚餐

### 3. 庐字号·山野庐山味 ⭐文化地标
- **地址**：牯岭镇电厂路3号（肯德基正对面）
- **人均**：65-75元
- **特色**：庐山首家三国主题饭店，江西非遗瓦罐汤工艺
- **招牌菜**：东吴石鸡煲、石鱼黄金蛋、石耳乌鸡瓦罐汤、黑毛香猪肉、现炒吊龙黄牛肉
- **特点**：严选山间食材、无预制菜、少油少盐健康做法
- **适合**：文化体验、研学团、家庭聚餐

### 4. 望庐说·本地菜馆 ⭐观景网红
- **地址**：牯岭镇隧道口（索道旁）
- **人均**：80-100元
- **评分**：4.5分
- **招牌菜**：板栗烧鸡、外婆家粉皮、炝鸭血、干锅笋、云雾茶香鸭、糖醋排骨、神仙凉面
- **亮点**：全景窗边山景位，傍晚观景绝佳
- **风格**：创意融合菜，口味偏清淡，改良版庐山特色菜
- **适合**：情侣约会、拍照打卡、不吃辣的游客

### 5. 庐人村·牯岭美食集
- **地址**：牯岭镇正街

### 6. 847别墅餐厅 ⭐氛围感

## 二、庐山必吃"三石一茶"

### 三石（必尝山珍）
| 名称 | 是什么 | 推荐吃法 |
|------|--------|----------|
| **石鸡** | 庐山溪涧野生蛙类（不是鸡！） | 红烧/清蒸，肉质细嫩 |
| **石鱼** | 体长寸许的小型野生鱼 | 石鱼炒蛋、石鱼汤 |
| **石耳** | 悬崖上的地衣植物 | 石耳炖土鸡、凉拌 |

### 云雾茶系列
- 云雾茶香骨：排骨用云雾茶汤慢煨，带兰花香气
- 云雾茶香鸭：望庐说的招牌创意菜
- 庐山云雾茶：中国十大名茶之一，可品尝或购买伴手礼

## 五、实用提示

- **物价说明**：山上物价比山下高，人均40-70能吃得不错
- **点菜注意**："时价"菜品务必先问清是按份还是按斤计价
- **善用团购**：大部分餐馆在美团/大众点评有套餐
- **避开拉客**：正街过于热情拉客的店需谨慎
- **错峰用餐**：高峰12:00-13:00 / 18:00-19:30
- **口味说明**：赣菜偏咸辣，不能吃辣记得说「微辣」或「免辣」
- **购物推荐**：天天平价超市（镇上最便宜）、陈氏茶饼世家（茶饼最好吃）
