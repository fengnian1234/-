整体风格定位：
请采用“新中式极简·禅意茶室”风格。整体视觉需要传达“有机、自然、宁静、轻奢”的感受，结合现代移动端/桌面端的极简高效 UI。既要有东方茶文化的温润感，又要有科技产品的清爽、干净、易用性。大量使用留白（Negative Space）和卡片式设计。

1. 配色系统（重点参考以下色值）：

主品牌色（关键操作、重要标识）： 墨绿 / 深林地绿（参考色值 #2C4335 或 #253D30）。这个绿色不能太鲜艳，要偏暗、偏灰，带有泥土和植物的自然质感。用于主按钮、头部导航、底部结算栏。

全局背景色： 暖米白 / 奶杏色（参考色值 #F8F5F1 或 #FDFCF9）。不要用纯白，要用这种微微偏暖的米色，营造纸张或棉麻的质感。

卡片/内容容器色： 纯白（或极浅的米白）。

辅助色/警示/标签底色： 淡卡其色 / 浅奶茶色（参考色值 #F0EBE1 或 #EFECE5）。用作状态标签（如“待接单”）、分类按钮、提示小面板的背景色。

文字颜色： 深碳黑（主标题，不能用纯黑，用深灰色）、中灰色（辅助信息，如 #8C8C8C）。

阴影： 使用极弱的柔和阴影，采用半透明的黑/灰色（如 rgba(0, 0, 0, 0.04)），突出卡片层级但不显脏。

2. 排版与字体：

字体类型： 现代无衬线体系（如系统默认的 PingFang SC、Roboto、Helvetica Neue）。

粗细与层级：

重点信息（如价格、订单号）： 大字号、加粗（Bold）。

状态文字： 中等粗细，颜色视状态而定（如待接单用深绿色，可领取用灰色）。

辅助信息： 细体、小字号、中灰色。

行间距： 适中偏宽，保持呼吸感。

3. 布局与轮廓：

形状特征： 全局大圆角。卡片、按钮、标签的圆角约为 8px 到 16px。没有锐利的直角。

结构（以移动端为主）：

顶部为沉浸式图片头或品牌头部，下方紧跟当前订单/取餐状态提示条。

中间内容区采用网格或列表式卡片排列（例如四个主要功能入口）。

底部为固定式 Tab 导航栏（4-5个图标）。

关键交互设计： 页面底部或侧边增加悬浮胶囊式按钮组（如移动端底部悬浮的购物车 ¥78 去结算，底色为墨绿，圆角药丸状）。

结构（商家后台桌面端）：

左侧固定导航栏 + 右侧主内容区 + 最右侧详情操作面板（若有必要）。

4. UI 组件细节：

状态标签（Pills）： 圆角药丸状（无边框），填充对应状态的底色（如“制作中”使用浅灰/浅米色底色），文字居中。

操作按钮：

主按钮： 墨绿背景，白色文字，无边框。

副按钮/取消按钮： 白色/米色背景，淡灰色边框，深色文字。

圆形加减按钮： 灰底白圈，或者极简的灰线边框圆圈。

头像/图标： 使用极简线条风格的图标，避免过于复杂的轮廓。头像是正圆形。

模态框（登录/授权）： 底部弹出式抽屉（Bottom Sheet），顶部有带 Logo 的半圆，按钮巨大且圆润（“拒绝”为白底灰框，“允许”为墨绿底白字）。

5. 品牌氛围强化：

情绪关键词： “云海之上”、“一杯此山茶”、“山中客”。设计中的文案要配合这种淡泊、自然的语境，界面中的加载状态或空状态，可以用茶叶、云雾、山脉的极简矢量插画来点缀，增加品牌调性。

质感模拟： 尽量减少高光反射，多使用哑光、磨砂或纸张质感的视觉感受。
