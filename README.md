- 高德/腾讯地图一键导航

### 🛎️ 快捷服务
- 15项快捷服务（续住/打扫/维修/叫醒/送餐/洗衣/叫车/行李寄存等）
- 分类清晰：客房服务、设施维修、前台服务
- 一键呼叫，即刻响应

### 📱 H5移动端页面
- 简洁、干净、典雅的设计风格

## 🚀 快速开始

### 1. 环境要求
- Python 3.10+
- pip

### 2. 安装依赖

```bash
cd d:/cc
pip install -r requirements.txt
```

### 3. 配置环境变量

```bash
# 复制配置文件
cp .env.example .env

# 编辑 .env，填写必要信息
```

**必须配置的项目**：
- `WECHAT_TOKEN`：微信公众号后台设置的Token
- `WECHAT_APP_ID` / `WECHAT_APP_SECRET`：公众号后台获取
- `ANTHROPIC_API_KEY`：（可选）启用AI智能对话

### 4. 初始化数据库并启动

```bash
# 初始化种子数据
python seed_data.py

# 启动服务
python run.py
```

服务启动后访问：
- 首页：`http://127.0.0.1:5000`
- 房型展示：`http://127.0.0.1:5000/rooms`
- 在线点餐：`http://127.0.0.1:5000/menu`
- 游玩攻略：`http://127.0.0.1:5000/travel`
- 快捷服务：`http://127.0.0.1:5000/services`

### 5. 接入微信公众号

1. 将服务部署到公网服务器（需要HTTPS域名）
2. 在微信公众号后台「开发 → 基本配置」中设置：
   - URL：`https://你的域名/wechat`
   - Token：与 `.env` 中的 `WECHAT_TOKEN` 一致
3. 提交验证，验证通过后即可接收用户消息

## 📋 微信公众号注册指南

### 第一步：选择账号类型

| 类型 | 年费 | 群发次数 | 微信支付 | 推荐场景 |
|------|------|---------|---------|---------|
| **服务号** | ¥300/年 | 4次/月 | ✅ 支持 | **推荐民宿使用** |
| 订阅号 | 免费 | 1次/天 | ❌ 不支持 | 媒体/自媒体 |
| 企业微信 | 免费 | - | ✅ 支持 | 企业内部管理 |

> 💡 **建议注册服务号**：支持微信支付、自定义菜单、模板消息等高级功能，适合民宿商业运营。

### 第二步：准备材料

- **企业/个体工商户**：
  - 营业执照
  - 法人身份证
  - 对公银行账户（或法人个人银行卡）
  - 运营者手机号 + 身份证 + 微信

- **个人**（只能注册订阅号）：
  - 身份证
  - 手机号
  - 已绑定银行卡的微信号

### 第三步：注册流程

1. 访问 **https://mp.weixin.qq.com**
2. 点击「立即注册」，选择账号类型
3. 填写基本信息（邮箱、密码）
4. 选择注册地「中国大陆」
5. 选择账号类型「服务号」
6. 主体信息登记（上传营业执照、填写法人信息）
7. 填写账号名称「云上归墅」、功能介绍
8. 提交审核（1-7个工作日）

### 第四步：认证（可选但推荐）

微信认证费用：¥300/次/年
- 认证后获得更多接口权限
- 账号名称带「V」标识，增加信任度

### 第五步：配置服务器

1. 登录公众号后台 → 开发 → 基本配置
2. 启用开发者密码(AppSecret)
3. 配置IP白名单（服务器IP）
4. 设置服务器配置：
   - URL：`https://你的域名/wechat`
   - Token：自定义一个字符串
   - EncodingAESKey：随机生成
   - 消息加解密方式：明文模式（先用明文调试）

## 🚢 部署到生产环境

### 使用 Gunicorn + Nginx（推荐）

```bash
# 安装 gunicorn
pip install gunicorn

# 启动
gunicorn -w 4 -b 127.0.0.1:8000 app:app
```

Nginx 配置示例：

```nginx
server {
    listen 80;
    server_name 你的域名;

    location /wechat {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
    }
}
```

> ⚠️ **重要**：微信要求服务器必须使用 **HTTPS**（443端口），建议使用 Let's Encrypt 免费证书或云服务商的SSL证书。

### 使用云服务（更简单）

也可以部署到：
- **腾讯云 CloudBase（云开发）**：与微信生态无缝集成
- **阿里云 ECS / 函数计算**
- **Railway / Render**（海外访问）
