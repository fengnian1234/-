🔌 预订管理 API

POST/api/booking/confirm
前台确认预订，解锁 AI 管家
参数: openid, guest_name, phone, platform, check_in, check_out, room_type
POST/api/booking/<id>/checkin
办理入住
参数: room_number
POST/api/booking/<id>/checkout
办理退房（触发好评推送倒计时）
POST/api/booking/check-ai
检查 AI 是否解锁
参数: openid
GET/api/booking/platforms
获取预订平台链接

⭐ 积分体系 API

GET/api/points/<openid>
获取积分概览（积分/等级/兑换列表/规则）
POST/api/points/earn
获取积分（签到/评价/分享等）
参数: openid, action
POST/api/points/redeem
兑换商品
参数: openid, item

🗄️ 数据库 Schema (10张表)

rooms · 房型表

字段类型说明
idInteger PK主键
nameString(100)房间名称
price / original_priceFloat价格 / 原价
capacity / bed_type / areaInteger / String人数 / 床型 / 面积
amenities / imagesJSON设施列表 / 图片URL列表
view_typeString(100)景观: 山景/云海/竹林

bookings · 预订表

字段说明
statusconfirmed → checked_in → checked_out → completed
ai_enabledAI管家是否解锁（确认预订后开启）
check_in_date / check_out_date入住/退房日期
checked_out_at退房时间（触发30分钟好评推送倒计时）
review_sent是否已推送好评提醒

其余8张表概览

表名用途
menu_categories / menu_items咖啡简餐菜单（分类+菜品）
orders点餐订单（微信支付）
travel_routes庐山游玩路线（景点/时长/难度）
food_recommends周边美食推荐（人均/必点/标签）

aggregated_orders多平台订单聚合（含佣金计算）
message_logs消息日志（AI对话持久化）
